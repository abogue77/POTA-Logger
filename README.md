POTA Hunter is a full featured windows and linux capable application specifically designed for POTA.  Please submit all issues and requests to to this git.

Features include:
  Flrig transeiver control integration.
  Live POTA Spots with internet connection.
  Scan POTA Spots with Flrig transceiver control.
  Easily visible highlighted rows for tuned stations and logged QSO's.
  Skip logged stations toggle on scan.
  Filter by band, mode, hide QRT.
  Pause live POTA spots updates.
  Click on station row fills out QSO log for fast logging.
  Park database loaded for park grid square and park name in log.
  Standard ADIF log format.
  Map for visual grid squares logged.
